# 科研驾驶舱

生成时间：2026-07-17

## 当前理论

战斗机甲不是单一贾克斯答案，而是德莱文保血、贾克斯中期承接、蕾欧娜/五费控制争一的数值壳。

## 假设
- H1 [支持] 官方S级不等于数值最强，只是高容错推荐集
  - 证据：模型分 vs 官方梯队 Spearman ρ=0.234；官方最高同口径为 无情连打贾克斯 2168分。
  - 下一步：继续找“官方配角当主C”与“官方未组壳”的高分异常。
- H2 [修正] 贾克斯是吃分桥，不是无条件终局答案
  - 证据：启动失败样本 21.3%；能启动样本 73.8%。排序后终局贾克斯已退出前列。
  - 下一步：采集最后两回合样本，分离站位、控制、同行、是否三星四个变量。
- H3 [强支持] 战斗机甲的颠覆点不是贾克斯，而是可继承的前中后期数值壳
  - 证据：战斗机甲：贾克斯 -> 蕾欧娜 100分；前中期可保留 贾克斯、德莱文、孙悟空、波比；中后期可保留 贾克斯、德莱文、孙悟空；主C装备继承较顺；终局上限高但依赖上9/五费二星；实战校准：贾克斯更适合作为中期吃分桥，后期转蕾欧娜/五费控制补争一上限
  - 下一步：验证德莱文开局、贾克斯中期、蕾欧娜终局的装备继承阈值。
- H4 [待证伪] 官方配角蕾欧娜可能是版本最高上限主C之一
  - 证据：后期候选第一：蕾欧娜2星 2634分，官方配角；5战斗机甲 / 3混沌战士 / 2灵能使 / 2护卫。
  - 下一步：测试蕾欧娜装备是否真能从德莱文/贾克斯平滑继承，还是模型被单体木桩误导。

## 下一轮实验
- [P1] 贾克斯启动率实验：变量=三星/水银/站位/对手控制；方法=每局记录最后3回合主C伤害、死亡时间、是否摸到后排；成功标准=启动率稳定 >65% 才给终局主C权重
- [P1] 蕾欧娜终局可行性：变量=上9速度、二星概率、装备继承；方法=用抽卡模型叠加经济曲线，测从5-1到6-1的成型率；成功标准=成型概率与实战吃鸡率同时高于贾克斯终局线
- [P2] 官方配角主C搜索：变量=官方非主C英雄 + 三件套 + 羁绊壳；方法=对所有官方配角跑反事实主C，筛出高分但低推荐率样本；成功标准=找出至少3套官方未命名的可复现路线
- [P2] 控制/站位惩罚模型：变量=近战、远程、AOE、控制免疫、第一目标可达性；方法=把战绩图伤害与木桩预期相除，拟合 matchupAccess；成功标准=同一阵容的预测误差下降到20%以内
